Как установить Opera в Windows

18.09.2024, 12:58


Здравствуйте! Opera - это один из популярных веб-браузеров. Мы уже рассказали, как установить Яндекс.Браузер, Chrome, Firefox, но мы так и не рассказали, как установить браузер Opera. Для этого вам даже не нужны права администратора!

ВНИМАНИЕ! Минимальная версия Windows для работы Opera - Windows 10.



Пункт №1. Перейдите на сайт https://www.opera.com/ru, нажмите по кнопке "Загрузить сейчас". Дождитесь окончания загрузки файла.



Пункт №2. Запустите скачанный файл.



Пункт №3. Нажмите "Принять и установить".



Пункт №4. Нажмите "Принять". Дождитесь окончания установки.



Пункт №5. Готово! По завершении установки браузер добавит себя на рабочий стол, предложит закрепить себя на панель задач и запустится. Осталось пройти первоначальную настройку, с ней справится каждый!
