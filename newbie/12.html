Как найти нужный файл в Windows с помощью проводника

23.09.2024, 22:35


Здравствуйте! Бывает, создали нужный файл, вышли и забыли где он находится. Для этого в Windows есть поиск! В этой инструкции я расскажу как его использовать.



Пункт №1. Откройте проводник. Для этого откройте "Пуск" и найдите "Проводник", затем запустите его.

Пункт №2. Найдите "Поиск". Он находится в правом верхнем углу проводника.

Пункт №3. Введите название вашего файла. Он будет найден, однако процесс поиска может занять некоторое время!

Следите за нашим сайтом, чтобы не пропускать новые инструкции для вас!
