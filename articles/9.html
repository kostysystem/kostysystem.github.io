Современный браузер для Windows XP

18.09.2024, 22:15


Здравствуйте! Я думаю вы слышали, а возможно, использовали Windows XP. Это операционная система, вышедшая в 2001 году. В своё время она стала революционной: темы, новый дизайн, стабильность и не только. В момент написания статьи, системе 23 года. Большинство программ закончили поддержку данной системы, включая браузеры. Казалось бы - всё! Однако, кто-то создал браузер на основе Firefox для Windows XP. И называется он - Mypal! Увы, в этом браузере нету поддержки русского языка. Он позволяет по сей день сидеть в интернете с Windows XP. Главное, чтобы ваш ретро-компьютер был подключён к интернету.

В этой статье я расскажу, как установить браузер Mypal на Windows XP.




Пункт №1. Откройте Internet Explorer.



Пункт №2. Перейдите по URL http://www.mypal-browser.org/download.html. В разделе "New version - (версия)" нажмите по "32bit".



Пункт №3. В появившимся окошке нажмите "Сохранить". Дождитесь окончания загрузки.



Пункт №4. Распакуйте скачанный ZIP-архив. Поместите файлы оттуда в любое место, желательно на другой том.



Пункт №5. Перейдите в папку. Найдите там файл mypal.exe. Запустите его.



Пункт №6. Mypal запустился! Вы можете создать ярлык на рабочем столе для файла mypal.exe, чтобы быстро запускать Mypal.
